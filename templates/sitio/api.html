Introducción

Esta API te permite interactuar con nuestro servicio para obtener datos, imagenes aleatorias de usuario, conversión sencilla de Text to speech, o el uso de widgets. Pronto irémos añadiendo más funcionalidades. 🔓
En la siguiente tabla encontrarás una lista de endpoints disponibles y sus ejemplos de uso.

Endpoints

Método Endpoint Descripción Acción Acción
GET https://blog-edu-tech.koyeb.app/foto-mujer Obtiene una foto aleatoria de una mujer.
GET https://blog-edu-tech.koyeb.app/foto-hombre Obtiene una foto aleatoria de un hombre.
